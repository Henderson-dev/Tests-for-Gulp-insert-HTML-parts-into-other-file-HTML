# Tests for Gulp insert HTML parts into other file HTML

## Dependencies

- Gulp
- Gulp File Include

## Install

```bash
$ yarn add gulp gulp-file-include -D
```

https://www.npmjs.com/package/gulp-html-import

## Run the gulp command

```bash
$ yarn gulp
```
